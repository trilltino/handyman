# Veo Video Scripts for XF Tradesmen

12 ready-to-produce video scripts for TikTok, YouTube Shorts, and Instagram Reels.

## Script Categories

### Pain Points (Build awareness)
1. `01_word_of_mouth_pain.md` - Relying on referrals
2. `02_google_invisible.md` - Being invisible on Google
3. `03_losing_to_competitors.md` - Competitors with websites winning

### Solutions (Show the benefit)
4. `04_triple_bookings.md` - Success story format
5. `05_website_works_while_you_sleep.md` - 24/7 lead generation
6. `06_first_page_transformation.md` - SEO transformation

### Educational (Add value)
7. `07_three_reasons_2026.md` - Why you need a website now
8. `08_handyman_near_me.md` - Customer search journey
9. `09_biggest_mistake.md` - Facebook vs Website

### Promotional (Direct CTA)
10. `10_72_hours_launch.md` - Speed of delivery
11. `11_329_investment.md` - Value proposition
12. `12_no_contracts.md` - Objection handling

## Each Script Contains
- Hook (0-3 sec attention grabber)
- Body (main content with voiceover)
- CTA (call to action)
- Visual directions for Veo AI
- Text overlay suggestions
- Music/tone guidance

## Posting Strategy
- Post 3-4x per week
- Rotate between categories
- Best times: 7-9am, 12-2pm, 6-8pm UK time
- Use trending sounds when relevant
- Add relevant hashtags: #tradesperson #handyman #smallbusiness #websitedesign #coventry

## Veo Tips
- Use the "Veo Prompt" at the bottom of each script
- Generate 2-3 variations per script
- Add your branding/logo in post-production
